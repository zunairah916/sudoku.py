Sudoku Generator & Solver

A Python application that generates Sudoku puzzles with a unique solution and solves them with animation. Built using Pygame with clean visuals and interactive features.

Features

Puzzle Generator: Randomly generates a 9×9 Sudoku puzzle ensuring only one solution.

Solver: Solves puzzles using backtracking with MRV (Minimum Remaining Value) for efficiency.

Animated Solving: Watch the solver fill the board step by step.

Interactive UI:

Press SPACE to animate the solver.

Press R to reset the board to the original puzzle.

Clean & Pleasant Visuals: Bold 3×3 box lines, highlighted cells during solving, smooth animations.
